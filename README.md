# URL shield - phishing URL detecter
AI-powered tool to detect phishing URLs in pakistan and worldwide.
## problems
Phishing attacks are common in pakistan -fake bank links,fake JazzCash pages,fake government sites.
## Solution
URL Shield analyzes any URL and tells you if it is SAFE or PHISHING.
## how it works
- Checks URL  length
- Detects suspicious words (login, verify, bank, free)
- Detects @ symbols
- Detects IP addresses instead of domains
- Detects excessive hyphens
